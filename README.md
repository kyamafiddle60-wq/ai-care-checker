# 🏥 AI Ready Checker - 介護施設版

介護施設のAI導入準備度を診断するStreamlitアプリケーション

## 📋 機能

- **無料診断30問**: 6カテゴリー × 5問の診断
- **カテゴリー別分析**: レーダーチャートで可視化
- **業界平均との比較**: あなたの施設と業界平均を比較
- **改善優先度**: スコアが低い順に改善提案を表示

## 🚀 起動方法

### 1. 仮想環境の有効化（推奨）

```bash
# 仮想環境を有効化
source venv/bin/activate
```

### 2. 依存関係のインストール

```bash
# 必要なパッケージをインストール
pip install -r requirements.txt
```

### 3. アプリの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501` でアプリが表示されます。

## 📁 プロジェクト構成

```
ai-care-checker/
├── app.py                 # メインアプリケーション
├── pages/                 # ページファイル
│   ├── 1_診断開始.py      # 診断フォーム
│   └── 2_診断結果.py      # 診断結果表示
├── modules/               # モジュール
│   ├── questions.py       # 質問データ
│   └── scoring.py         # スコアリング機能
└── requirements.txt       # 依存関係
```

## 🎯 使い方

1. ホームページから「無料診断を始める」ボタンをクリック
2. 30問の質問に回答
3. 診断結果を確認
4. カテゴリー別スコアと改善提案を確認

## 📊 カテゴリー

- **ビジネス準備度**: 経営陣の理解、予算、ROI期待など
- **データ準備度**: 記録のデジタル化、データ品質など
- **組織準備度**: 職員のITリテラシー、研修体制など
- **技術準備度**: IT環境、セキュリティ、ネットワークなど
- **プロセス準備度**: 業務標準化、効率化意識など
- **コンプライアンス準備度**: 個人情報保護、法令遵守など
